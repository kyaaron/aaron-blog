Welcome to my new blog! I've built this using **React**, **Vite**, and **Tailwind CSS v4**.

## Why GitHub as a CMS?

Using GitHub as a CMS is a great way to:
- Keep everything in version control.
- Write in Markdown (which we all love).
- Host for free on GitHub Pages.
