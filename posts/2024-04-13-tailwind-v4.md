Tailwind CSS v4 is finally here, and it brings some amazing improvements to the developer experience.

## What's New?

- **Zero-config Engine**: Most of the configuration now lives directly in your CSS files.
- **Native Cascading Layers**: Better integration with CSS standards.
- **Improved Performance**: Faster builds and smaller bundles.

The typography plugin (`@tailwindcss/typography`) is still essential for rendering beautiful markdown content, like what you're reading right now!

> "Tailwind CSS makes styling feel like play."

I'm excited to keep using it in all my projects.
