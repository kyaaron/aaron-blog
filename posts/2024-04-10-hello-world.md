I finally did it. I fully uninstalled Windows 11 from my gaming PC. I didn’t install Linux side by side with Windows. No dual booting. Linux is now 100% the only operating system on my gaming PC.

It took awhile to finally do it. I started thinking about replacing Windows, at least in part, several months ago after reading about more advertisements and forced AI integration into Windows 11. Also, apparently Windows 11 sends a lot of user data back to Microsoft and it forces you to create a Microsoft account when installing (why does everything require an account now anyways?). I’m just not interested in that. At some point, when does my PC become essentially Microsoft’s PC where they control my experience?

Linux has come a long way. The reputation I think Linux has is dated. “Linux is hard”, “Linux is confusing”, “Linux doesn’t work well with well-known software”, “Good luck getting hardware and software drivers to work on it”, “You can’t game on Linux”, etc. I could go on. If you look at gaming numbers between Windows, Linux, and MacOS, Windows still has about a 95% market share, while Linux has 2.89% of the market share (at least among Steam users). That’s about 4 million gamers playing on Linux out of a reported 132 million on Steam.

The honest truth though is that Linux has come a long way in the past several years. A lot of that is thanks to Steam. Steam released the Steam Deck a few years ago which runs on Linux. The portable handheld gaming console has been immensely popular. Because it’s using Linux, there had to be tools created to translate all the games made for Windows into a format that could be played on Linux. Not all of this is because of the Steam Deck, but it certainly helped.

So finally I decided to pull the trigger. A friend of mine pinged me on Discord and told me he had just moved from Windows 11 to Bazzite OS. Bazzite is a distribution of Linux that is specialized for gaming. I asked him how hard it was and did his games play. He said it was very easy to move over, and pretty much everything was playing great for him (there were some issues with multiplayer games that use Anticheat software but he was working on it). That was what I needed to hear.

I went to Bazzite’s website and downloaded the ISO file for the OS onto a flash drive. I formatted it to be bootable from my PC’s UEFI, then pulled up directions to install it on my computer. I took all my important files on my computer (which were mostly already in OneDrive) and backed them up into my Google Cloud. Then I went into my UEFI, updated the boot order to boot from my USB containing the Bazzite ISO file, and started installing.

It didn’t take long to install. Once it was finished, I restarted my computer, removed the USB, and waited to see what would happen. I got to the new Bazzite login screen, and was able to log in with no problem! Bazzite automatically came with Steam. Once I logged into my Steam account, all my games appeared. I had to reinstall them, so I started with a few smaller games to test it out. Valheim worked with no problem, and Phasmophobia mostly works. The single player ghost hunts had no issue. The multiplayer servers might be a bit spotty but I haven’t tried them yet.

I was able to install Vivaldi for my browser and Discord with no issue. Linux versions of all the popular software we use for personal PCs are out there and readily available.

All in all, I’m happy with the switch. I cannot promise you that every single game out there works, but there are plenty of other people who have run into issues and found ways to manually install the necessary updates and workarounds to have games run correctly. And I should say, I’m not anti-Microsoft. I have no ill-will against the company myself. But I decided that I didn’t want to have Windows 11 any longer or be dependent on the Windows ecosystem. I’ve heard rumors of Windows 12 and more AI stuff they will be pushing into it, and I’m not particularlly interested in that being the experience I get when I log into my PC.

Another note I should add is my PC is now six years old. I haven’t updated anything on it since I built it. So I’m not running the newest AAA games out there. And I don’t have any interest in updating my PC, because thanks to AI all the hardware prices are too expensive. Eventually Windows was going to be taxing on my hardware.

I’ve been on Linux now for about a week. I love it. I’m glad I moved over. It’s not as hard as people make it seem sometimes. It’s 100% free, easy to install, and is MUCH better for gaming than it used to be. One other perk is the install speed has drastically increased. I was installing games at around 3–5 Mbps on Windows, and now I’m installing games on the same internet at 30–35 Mbps speed. I looked up why, and apparently Bazzite is more optimized for downloading games and updates than Windows!
